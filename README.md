# WEB FIX - Consultoria Tech 🚀

Projeto de desenvolvimento web purista desenvolvido em parceria com a **Pichau**, focado em fundamentos de estruturação e semântica.

## 🎓 Contexto Acadêmico
Este projeto é o resultado do módulo de **Desenvolvimento Web** do **IOS - Instituto da Oportunidade Social** (Unidade Santana). O desafio consistiu em criar uma interface profissional utilizando o mínimo de recursos de estilização modernos, focando no domínio total da estrutura HTML5.

## 🛠️ Tecnologias e Restrições Técnicas

O projeto seguiu diretrizes rigorosas ("purismo técnico") para garantir que a base do desenvolvimento front-end fosse consolidada:

### **Estrutura (HTML5 Semântico)**
Utilização avançada de tags semânticas para acessibilidade e SEO:
- `<header>`, `<nav>`, `<main>`, `<section>`, `<article>`, `<footer>`
- **Uso de Dados:** Implementação de `<figure>` e `<figcaption>` para imagens e `<address>` para informações de contato.

### **Estilização (CSS Restrito)**
A estilização foi limitada a propriedades fundamentais, exigindo soluções criativas para o layout:
1. `background-color` (Cores de fundo e identidade visual)
2. `color` (Tratamento tipográfico)
3. `text-align` (Alinhamento de elementos)
4. `height` (Controle de dimensões)
5. `list-style: none` (**Nota:** Utilizada sob autorização especial do professor para garantir a limpeza visual do menu de navegação).

## 📐 Decisões de Arquitetura e Design

### **Layout de Transição (Tabelas)**
Devido à proibição de propriedades modernas como `flexbox` ou `display`, utilizei a tag `<table>` para estruturar o cabeçalho e garantir que o logotipo e o menu estivessem posicionados corretamente (esquerda/direita), simulando a organização de sites clássicos e robustos.

### **Seletores: Classes vs IDs**
Optei por utilizar exclusivamente **classes** para a estilização. Embora o uso de `id` fosse permitido, decidi não utilizá-los por não haver necessidade técnica de especificidade única, mantendo o código mais limpo, reutilizável e seguindo as melhores práticas de manutenção.

### **Proibições Respeitadas**
- ❌ **Sem JavaScript:** Interface 100% funcional apenas com HTML/CSS.
- ❌ **Sem Frameworks:** Ausência de Bootstrap ou similares.
- ❌ **Sem CSS Moderno:** Zero uso de Flexbox, Grid ou posicionamento absoluto/relativo.
